# Frenzy & Fiery Fries — Landing Page

React (JSX) + Tailwind CSS landing page inspired by the layout/sections of:
- `https://grabordine.co.uk/`
- `https://www.fireaway.co.uk`

## Requirements
- Node.js 18+ (you have Node installed already)

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed in the terminal.

## Notes (Windows folder name with `&`)
This project lives in a folder containing `&` (`Frenzy & Fiery Fries`), which can break some Windows command wrappers.
The `package.json` scripts call Vite via `node` directly to avoid that issue.
